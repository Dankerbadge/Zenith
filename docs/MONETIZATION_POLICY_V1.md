# Monetization Policy V1

Last updated: 2026-02-04

## Free Tier (Guaranteed)

Zenith free tier includes:

- Core logging (food, water, workout, active rest, weight, runs)
- Stats and trend views
- Winning days, streaks, and ranks
- Core social features (friends, feed, clubs baseline)
- Basic AI insights (optional, OFF by default)
- Export/restore controls available in app

## Premium Candidates (Deferred)

Premium is limited to optional enhancements only:

- Advanced analytics packs
- Optional richer AI phrasing layers
- Cross-device sync expansions
- Advanced team/admin tooling

## Cost Guardrails (Non-Negotiable)

- No required per-use paid APIs in v1
- No paid dependency may block core discipline workflows
- Any paid integration must be behind an explicit toggle/paywall
- Optional paid integrations default to OFF

## Disallowed for V1

- Mandatory cloud AI usage for core features
- Paywall gating of essential logging/stats/streak flows
- Hidden recurring infrastructure costs in baseline product path

## Release Gate Check

Before App Store submission:

- Free tier list above must remain true
- Premium scope must remain optional and clearly labeled
- Support and legal links must be live and public
