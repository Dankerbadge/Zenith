Getting Started

Set up Zenith in under ten minutes.

1) Create account + identity

  • Use your primary email
  • Set base goals (protein, water, active rest)
  • Choose week start and units

2) Log your first day

  • Log at least one qualifying action
  • Check Home for Winning Day status
  • Open Stats after two to three days for trend context

3) Keep momentum simple

  • Use Quick Log as default entry point
  • Follow “one thing left” guidance when shown
  • Focus on repeatable, not perfect, days

Need help now?

If setup feels off, contact support and include app version + device model.
